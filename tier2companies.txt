### üß© Example Problem & Solution

**Reverse a Linked List** (frequent Tier-2 DS challenge):

```java
public class ListNode { int val; ListNode next; ListNode(int v){ val=v; } }

public ListNode reverseList(ListNode head) {
    ListNode prev = null;
    ListNode curr = head;
    while (curr != null) {
        ListNode next = curr.next;
        curr.next = prev;
        prev = curr;
        curr = next;
    }
    return prev;
}
```
